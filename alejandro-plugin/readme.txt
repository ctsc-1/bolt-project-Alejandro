    Plugin de gestion des préférences utilisateurs pour WordPress.

    == Description ==

    Ce plugin permet aux utilisateurs de gérer leurs préférences et paramètres.

    == Installation ==

    1. Téléchargez le plugin
    2. Décompressez le fichier
    3. Placez le dossier dans /wp-content/plugins/
    4. Activez le plugin dans l'administration WordPress
